# BatMans-Towing LLC Website

A minimalist static website built with HTML and CSS, featuring a yellow and black color palette. This site displays BatMans-Towing LLC staff rosters including Owner, TWL Staff, Managers, and Operators, as well as interests in games like Motor Town, ETS2, and ATS, and tools like the Trucky client used for BatMans Trucking LLC (subsidiary), in various formats: lists, tables, and cards.

## Features

- **List View**: Displays items in a simple unordered list.
- **Table View**: Presents data in a structured table format.
- **Card View**: Shows information in visually appealing cards.
- **Interactive Elements**: Includes a toggle button to show/hide a list.

## Color Palette

- Background: Black (#000000)
- Text: Yellow (#FFFF00)
- Accents: Yellow (#FFFF00)
- Secondary Background: Dark Gray (#333333)

## Files

- `index.html`: Main HTML structure
- `style.css`: CSS styles with minimalist design
- `script.js`: JavaScript for interactivity
- `README.md`: This documentation file

## How to View

Open `index.html` in any modern web browser to view the website.

## Customization

To add your own information:
1. Edit the content in `index.html` within the respective sections.
2. Modify styles in `style.css` if needed.
3. Enhance interactivity in `script.js`.
